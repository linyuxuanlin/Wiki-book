# 用 reveal.js 制作幻灯片

# 快捷键

  • 下一张幻灯片：空格
  • 方向选择幻灯片：方向键
  • 总览视图：Esc
  • 演讲者视图：S
  • 暂停演讲 / 黑屏：V/B/.

# PDF 导出

在地址后加 ?print-pdf，例如 http://localhost:8000/?print-pdf
