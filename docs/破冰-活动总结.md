# 破冰 - 活动总结

## 出现的问题

1. 彩排流程不充分
2. 工作人员未做好自己的本职工作，没有团队意识，时间观念不强
3. 规则制定不明确，出现很多问题未考虑进来
4. 某些游戏流程过于复杂，导致部分活动人员出现困倦的情绪
5. 拍合照角度未提前安排，没把所有人都拍进去
6. 活动摄像 / 摄影成片质量不佳，拍摄者未经过培训
